## v1.0.0 (2026-05)

**최초 릴리스**

### 추가된 시스템

- 튜토리얼 시스템 — ScriptableObject 기반 노드, 저장/복원
- 조건 그래프 — AND / OR / NOT 노드 트리
- 스탯 모디파이어 — Flat / PercentAdd / PercentMultiply 3단계 계산
- 포뮬러 시스템 — 재귀 하강 파서, 기획자 수정 가능 수식
- 커브 데이터 — AnimationCurve 기반 밸런스 테이블
- 시퀀스 시스템 — 선언적 연출 스텝 체인
- 카메라 디렉터 — Zoom / Shake / Move 액션 시퀀스
- 버프/디버프 — Duration, Stack, Tick 자동 관리
- 퀘스트 시스템 — Kill / Talk / Collect 이벤트 기반
- 스폰 시스템 — 가중치 랜덤, 최대 생존 수 관리
- 행동 기록/재생 — InputFrame 타임스탬프 기록 및 재생
- 시간 되감기 — IRewindable 스냅샷 히스토리
