**AchUtils** は、Unity のゲーム開発で毎回繰り返されるシステムを、独立したインスタンスとしてパッケージ化したコレクションです。

チュートリアル、条件グラフ、スタットモディファイアなど 12 のシステムがそれぞれ独立して動作するため、必要なものだけを選んで使えます。

## 特徴

- **独立インスタンス** — ほとんどのランタイムシステムは純粋な C# インスタンスとして生成し、コルーチン・Transform・Instantiate が必要な部分のみコンポーネントを使用
- **ScriptableObject ベース** — データとロジックを分離し、プランナーが直接編集
- **`[SerializeReference]` ポリモーフィズム** — ノード、エフェクト、ステップをインスペクターから直接追加
- **反復コードの削減** — ランタイムシステムに加えて、コレクション・Unity オブジェクト・UI・文字列の拡張メソッドを提供
